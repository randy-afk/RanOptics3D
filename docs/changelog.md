# Changelog

---

## v1.0.0

*Initial release.*

- Interactive 3D HTML lattice viewer with in-browser control panel
- Tao/Bmad, ELEGANT, MAD-X, and xsuite backend support
- Color-coded element geometry (dipoles, quads, sextupoles, kickers, RF cavities, solenoids)
- Solenoid helix geometry
- Beampipe, ground plane, and axes gizmo overlays
- σ tube beam envelope overlay
- Element highlighting by name or wildcard pattern
- Floating annotation labels by element pattern
- Camera presets (Iso, Top, Side, Front) and PNG screenshot export
- Independent X/Y/Z axis scaling
- Twiss Inspector: click-to-set s-range, optics popup (β, σ, η, orbit, phase advance)
- Selected Element panel: name, type, length, s, K1/K2/angle/voltage
- Multi-universe overlay support
- PySide6 GUI with live log
