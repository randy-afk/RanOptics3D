# Navigation & Controls

The 3D scene is fully mouse-driven.

---

## Mouse Controls

| Action | Control |
|---|---|
| **Rotate** | Left drag |
| **Pan** | Right drag |
| **Zoom** | Scroll wheel |
| **Select element** | Click — populates the Selected Element panel and enables Twiss Inspector range setting |

---

## Tips

!!! tip "Flat lattice rings"
    Use the **Aspect** sliders to exaggerate the Y scale. This makes the σ tube
    and vertical elements visible without needing to zoom in aggressively.

!!! tip "Finding a specific element"
    Use **Highlight** with a wildcard (e.g. `QD*`) to locate elements visually,
    then **Annotate** the same pattern to add labels for screenshots.

!!! tip "Screenshot workflow"
    1. Position the camera or use a preset.
    2. Annotate the elements you want labeled.
    3. Click **Screenshot** — PNG is saved automatically.
